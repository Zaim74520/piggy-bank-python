# Основы синтаксиса Python - проект «Виселица» (Hangman)
## Описание проекта
Это игра «Виселица», созданная в рамках изучения Python.

## Основные возможности проекта

### Функционал игры
* Выбор случайного слова
* Отгадывание букв по одной
* Подсчёт оставшихся попыток
* Отображение прогресса

### Использованные технологии
* Python 3.x
* Базовые конструкции языка
* Работа с файлами
* Обработка пользовательского ввода
